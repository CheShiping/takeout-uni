## 教程概览

本教程以 takeout-uni 项目为蓝本，从零到一讲解如何使用 uni-app 开发一个完整的外卖点餐系统，覆盖 H5、微信小程序双端。适合有 Vue 基础、想学习 uni-app 跨平台开发的同学。

### Part 1 · 基础篇

学习项目环境搭建、架构设计、网络请求封装、状态管理和类型系统设计，为后续开发打下坚实基础。

[开始学习 →](/tutorials/01-项目概述与环境搭建)

### Part 2 · 核心功能篇

实现首页、店铺、食品详情、购物车、订单等核心业务功能，掌握 uni-app 组件开发和页面交互。

[开始学习 →](/tutorials/06-首页开发)

### Part 3 · 进阶篇

学习地址管理、用户认证、评价系统、通用组件、跨平台适配、样式主题和构建发布等进阶内容。

[开始学习 →](/tutorials/11-地址管理)

## 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| **前端框架** | Vue 3 + TypeScript | 响应式 UI 开发 |
| **跨平台框架** | uni-app 3.x | 一套代码多端运行 |
| **状态管理** | Pinia | 全局状态管理 |
| **网络请求** | Axios | HTTP 请求封装 |
| **构建工具** | Vite | 快速开发构建 |
| **CSS 预处理器** | SCSS | 样式开发 |

| 路径 | 适合人群 | 预期收获 |
