# 第九章：购物车模块

## 9.1 购物车数据模型

购物车使用两种数据结构：

1. **CartFoodItem**（前端展示用）：继承 `FoodItem`，增加购买数量等字段
2. **CartItem**（后端同步用）：精简结构，只包含订单所需字段

```typescript
// 前端展示
interface CartFoodItem extends FoodItem {
  buyNum: number
  specId?: string
  specName?: string
}

// 后端同步
interface CartItem {
  id: string
  userId: string
  shopId: string
  foodId: string
  specId: string
  foodName: string
  foodImage: string
  specName: string
  price: number
  quantity: number
  subtotal: number
}
```

### 类型转换函数

```typescript
// src/models/cart.types.ts
export function cartFoodItemToCartItem(item: CartFoodItem): CartItem {
  let specId = item.specId || ''
  let specName = item.specName || ''
  if (!specId && item.specs?.length > 0) {
    specId = item.specs[0].id
    specName = item.specs[0].name
  }
  return {
    id: '', userId: '', shopId: item.shopId, foodId: item.id,
    specId, foodName: item.name, foodImage: item.image,
    specName, price: item.price, quantity: item.buyNum,
    subtotal: item.price * item.buyNum,
    createTime: new Date().toISOString()
  }
}
```

## 9.2 CartStore 设计

购物车状态存储在 `cart.store.ts` 中，使用 `Record<string, CartFoodItem>` 以食品 ID 为 key：

```typescript
export const useCartStore = defineStore('cart', () => {
  const choosePrice = ref<number>(0)
  const buyFoods = ref<Record<string, CartFoodItem>>({})

  // 计算属性
  const totalItems = computed(() => Object.keys(buyFoods.value).length)
  const totalPrice = computed(() => MathUtil.toFixed(choosePrice.value, 2))
  const buyFoodList = computed(() => Object.values(buyFoods.value))

  function addToCart(item: CartFoodItem): void {
    const existingItem = buyFoods.value[item.id]
    if (existingItem) {
      existingItem.buyNum += item.buyNum || 1
      choosePrice.value = MathUtil.addition(choosePrice.value, MathUtil.multiply(item.price, item.buyNum || 1))
    } else {
      const newItem = { ...item, buyNum: item.buyNum || 1 }
      buyFoods.value[item.id] = newItem
      choosePrice.value = MathUtil.addition(choosePrice.value, MathUtil.multiply(item.price, newItem.buyNum))
    }
  }

  function decreaseQuantity(foodId: string): void {
    const item = buyFoods.value[foodId]
    if (item) {
      if (item.buyNum > 1) {
        item.buyNum -= 1
        choosePrice.value = MathUtil.subtraction(choosePrice.value, item.price)
      } else {
        removeFromCart(foodId)  // 数量为 0 自动移除
      }
    }
  }
})
```

### 价格精度处理

购物车使用 `MathUtil` 工具类避免浮点数精度问题：

```typescript
// MathUtil.ts
static addition(a: number, b: number): number {
  const factor = MathUtil._getFactor(a, b)
  return (a * factor + b * factor) / factor
}

static multiply(a: number, b: number): number {
  const aStr = a.toString()
  const bStr = b.toString()
  const aDec = aStr.includes('.') ? aStr.split('.')[1].length : 0
  const bDec = bStr.includes('.') ? bStr.split('.')[1].length : 0
  const factor = Math.pow(10, aDec + bDec)
  return (Number(aStr.replace('.', '')) * Number(bStr.replace('.', ''))) / factor
}
```

## 9.3 useCart 组合式函数

`src/hooks/useCart.ts` 提供购物车的业务层封装：

```typescript
export function useCart() {
  const cartStore = useCartStore()

  const totalItems = computed(() => cartStore.totalItems)
  const totalPrice = computed(() => cartStore.totalPrice)
  const buyFoodList = computed(() => cartStore.buyFoodList)

  function addToCart(item: CartFoodItem) {
    cartStore.addToCart(item)
    ToastUtil.showSuccess('已加入购物车')  // 自动提示
  }

  function clearCart() {
    cartStore.clearCart()
  }

  return { totalItems, totalPrice, buyFoodList, addToCart, removeFromCart, increaseQuantity, decreaseQuantity, getBuyNum, clearCart }
}
```

Hook 的职责是：**Store 操作 + UI 反馈**，页面直接调用 Hook 即可。

## 9.4 CartService 后端同步

```typescript
export class CartService {
  /** 添加到购物车 */
  static async addToCart(request: AddCartRequest): Promise<CartItem> {
    const response = await httpUtil.post<CartItem>(
      API_CONFIG.PATHS.CART_ADD,
      request as unknown as Record<string, unknown>
    )
    return response.data
  }

  /** 同步购物车到服务器 */
  static async syncCart(request: CartSyncRequest): Promise<boolean> {
    const response = await httpUtil.post<boolean>(API_CONFIG.PATHS.CART_SYNC, request)
    return response.data || false
  }
}
```

## 9.5 购物车不持久化的设计

购物车状态（`cart.store.ts`）**没有配置持久化**，页面刷新后数据丢失。这是一个有意的设计决策：

- 用户在一次会话中完成点餐 → 下单流程
- 避免持久化带来的数据过期问题
- 后端有独立的购物车接口，可以做数据同步

## 9.6 购物车流程总结

```
用户选择食品 → FoodSpecDrawer 选择规格
    → BuyNumControl 设置数量
    → useCart().addToCart() 加入本地购物车
    → CartStore 更新状态和价格
    → 跳转到订单确认页
    → CartService.syncCart() 同步到后端
```
