# 第十章：订单流程

## 10.1 订单状态流转

订单的完整生命周期：

```
创建订单(unpaid) → 支付(paid) → 配送(delivering) → 完成(completed)
                    ↓
                取消(cancelled)
```

对应的枚举定义：

```typescript
enum OrderStatus {
  UNPAID = 'unpaid',
  PAID = 'paid',
  DELIVERING = 'delivering',
  COMPLETED = 'completed',
  CANCELLED = 'cancelled'
}
```

```typescript
async function submitOrder() {
  if (!addressId.value) {
    ToastUtil.showMessage('请选择收货地址')
    return
  }

  const request: CreateOrderRequest = {
    shopId: cartStore.chooseShop,
    orderType: cartStore.orderType,
    addressId: addressId.value,
    tableNumber: '',
    remark: remark.value,
    cartItemIds: cartStore.buyFoodList.map(item => item.id)
  }

  const orderId = await OrderService.createOrder(request)
  uni.redirectTo({ url: `/pages/order/payment?orderId=${orderId}` })
}
```

## 10.3 支付页面

`pages/order/payment.vue` 处理支付流程。

### PaymentService

```typescript
export class PaymentService {
  /** 创建支付 */
  static async createPayment(orderId: string, method: string): Promise<PaymentOrder> {
    const response = await httpUtil.post<PaymentOrder>(
      API_CONFIG.PATHS.PAYMENT_CREATE, undefined, { orderId }
    )
    return response.data
  }

  /** 轮询等待支付结果 */
  static async waitForPayment(paymentId: string, timeout: number = 30): Promise<PaymentStatus> {
    const startTime = Date.now()
    const maxTime = startTime + timeout * 1000

    while (Date.now() < maxTime) {
      const status = await PaymentService.getPaymentStatus(paymentId)
      if (status === PaymentStatus.SUCCESS || status === PaymentStatus.FAILED) {
        return status
      }
      await new Promise(resolve => setTimeout(resolve, 1000))
    }
    return await PaymentService.getPaymentStatus(paymentId)
  }
}
```

支付流程：创建支付 → 轮询状态 → 成功跳转 / 失败提示。

### 订单筛选

```typescript
function filterOrdersByTab(orders: Order[], tabType: OrderTabType): Order[] {
  switch (tabType) {
    case 'PENDING':
      return orders.filter(o =>
        o.status === OrderStatus.UNPAID ||
        o.status === OrderStatus.PAID ||
        o.status === OrderStatus.DELIVERING
      )
    case 'COMPLETED':
      return orders.filter(o => o.status === OrderStatus.COMPLETED)
    case 'CANCELED':
      return orders.filter(o => o.status === OrderStatus.CANCELLED)
    default:
      return orders
  }
}
```
