# 第一章：项目概述与环境搭建

## 1.1 项目介绍

takeout-uni 是一个基于 **Vue 3 + TypeScript + uni-app 3.x** 的跨平台外卖点餐系统，支持同时发布到 H5、微信小程序和 App 三个平台。

### 核心功能

- **首页**：Banner 轮播、分类入口、热销榜单、好评推荐
- **点餐**：食品分类浏览、规格选择、购物车管理
- **订单**：下单确认、支付、订单列表与详情
- **地址**：收货地址的增删改查
- **个人中心**：登录注册、个人资料、订单查看
- **评价**：星级评分、文字评价、图片上传

### 技术选型

| 技术 | 说明 |
|------|------|
| Vue 3 | 前端框架，使用 Composition API |
| TypeScript | 类型安全的 JavaScript 超集 |
| uni-app 3.x | 跨平台框架，一套代码多端运行 |
| Pinia | 状态管理，替代 Vuex |
| Axios | HTTP 请求库 |
| Vite | 构建工具，极速 HMR |
| SCSS | CSS 预处理器 |
| uview-pro | UI 组件库 |

## 1.2 开发环境准备

### 必装软件

1. **Node.js** (>= 16)
   - 下载地址：https://nodejs.org/
   - 安装后验证：
   ```bash
   node -v
   npm -v
   ```

2. **HBuilderX**（推荐，用于真机调试和 App 打包）
   - 下载地址：https://www.dcloud.io/hbuilderx.html
   - 安装 App 开发版

3. **微信开发者工具**（微信小程序开发必需）
   - 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 安装后需要在设置中开启"服务端口"

### 推荐编辑器

- **VS Code** + 以下插件：
  - Volar（Vue 3 支持）
  - uni-app-schemas（pages.json 智能提示）
  - ESLint（代码规范，可选）

## 1.3 创建项目

本项目使用 Vite 创建 uni-app Vue3 + TypeScript 项目：

```bash
# 创建项目（使用 uni-app 官方模板）
npm create uni-app@latest takeout-uni -- --template vue3-ts

# 进入项目目录
cd takeout-uni

# 安装依赖
npm install
```

### 安装额外依赖

```bash
# 状态管理
npm install pinia pinia-plugin-persistedstate

# HTTP 请求
npm install axios

# UI 组件库
npm install uview-pro
```

│   ├── services/         # 业务服务层（9 个服务类）
│   ├── stores/           # Pinia 状态管理

│   ├── utils/            # 工具函数

## 1.5 首次运行

### H5 开发模式

```bash
npm run dev:h5
```

启动后在浏览器访问 `http://localhost:5173`，可以看到项目的启动页（Splash），随后自动跳转到主页面。

### 微信小程序开发模式

```bash
npm run dev:mp-weixin
```

编译完成后，打开微信开发者工具，导入 `dist/dev/mp-weixin` 目录即可预览。

### 构建生产版本

```bash
# H5 生产构建
npm run build:h5

# 微信小程序构建
npm run build:mp-weixin
```
