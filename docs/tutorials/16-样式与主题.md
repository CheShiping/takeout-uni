# 第十六章：样式与主题

## 16.1 uni.scss 全局变量

`src/uni.scss` 是 uni-app 的全局样式变量文件，所有页面自动引入，无需手动 import。

```scss
/* uView Pro 样式 */
@import 'uview-pro/theme.scss';

/* 行为相关颜色 */
$uni-color-primary: #007aff;
$uni-color-success: #4cd964;
$uni-color-warning: #f0ad4e;
$uni-color-error: #dd524d;

/* 文字基本颜色 */
$uni-text-color: #333;
$uni-text-color-inverse: #fff;
$uni-text-color-grey: #999;
$uni-text-color-placeholder: #808080;

/* 背景颜色 */
$uni-bg-color: #ffffff;
$uni-bg-color-grey: #f8f8f8;
$uni-bg-color-hover: #f1f1f1;
$uni-bg-color-mask: rgba(0, 0, 0, 0.4);

/* 边框颜色 */
$uni-border-color: #c8c7cc;

/* 文字尺寸 */
$uni-font-size-sm: 12px;
$uni-font-size-base: 14px;
$uni-font-size-lg: 16px;
```

在任何组件中可以直接使用这些变量：

```scss
.title {
  color: $uni-text-color;
  font-size: $uni-font-size-lg;
  background-color: $uni-bg-color;
}
```

## 16.2 品牌色系

项目的品牌主色是 `#FF6B35`（橙色），贯穿整个应用：

```scss
/* 品牌色使用示例 */
.brand-text {
  color: #FF6B35;
}

.brand-bg {
  background-color: #FF6B35;
}

.brand-gradient {
  background: linear-gradient(135deg, #FF6B35, #FF8F5E);
}

/* 品牌色变体 */
.brand-light {
  color: #FF8F5E;  /* 浅色变体 */
}

.brand-dark {
  color: #E55A2B;  /* 深色变体 */
}
```

品牌色在以下场景使用：
- 底部 Tab 激活态
- 按钮背景
- 价格文字
- 选中状态指示器
- 启动页渐变背景

## 16.3 Scoped SCSS

所有组件使用 `<style lang="scss" scoped>` 实现样式隔离：

```html
<style lang="scss" scoped>
.container {
  padding: 16px;
  background-color: #ffffff;
}

.title {
  font-size: 18px;
  font-weight: bold;
  color: #333;
}
</style>
```

`scoped` 属性会给每个选择器添加唯一属性标记，确保样式不会泄漏到其他组件。

## 16.4 全局样式

在 `App.vue` 中定义全局基础样式：

```html
<style lang="scss">
page {
  font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto,
    'Helvetica Neue', Arial, sans-serif;
  color: #333;
  background-color: #f5f5f5;
  font-size: 14px;
}

.safe-area-bottom {
  padding-bottom: constant(safe-area-inset-bottom);
  padding-bottom: env(safe-area-inset-bottom);
}
</style>
```

注意：`App.vue` 的样式**不使用 scoped**，因为它需要影响所有页面。

## 16.5 rpx 响应式布局

uni-app 推荐使用 `rpx`（responsive pixel）作为尺寸单位：

```scss
.card {
  width: 750rpx;          // 设计稿满宽（iPhone 6 为基准）
  padding: 24rpx;
  margin-bottom: 20rpx;
  border-radius: 16rpx;
}

.avatar {
  width: 80rpx;
  height: 80rpx;
  border-radius: 50%;
}
```

**换算规则**：`1rpx = 0.5px`（在 iPhone 6 上），在其他设备上自动按比例缩放。

### 实际项目中的混合使用

本项目中 `rpx` 和 `px` 混合使用：
- 大尺寸（容器宽高、间距）用 `rpx`
- 小尺寸（字体、边框、图标）用 `px`

```scss
.section {
  margin: 0 16px 12px;        // px 用于页面边距
  padding: 16px;
  border-radius: 12px;
}

.food-name {
  font-size: 14px;             // px 用于字体
  line-height: 1.4;
}
```

## 16.6 常用布局模式

### Flex 布局（最常用）

```scss
/* 水平居中 */
.flex-center {
  display: flex;
  align-items: center;
  justify-content: center;
}

/* 两端对齐 */
.flex-between {
  display: flex;
  align-items: center;
  justify-content: space-between;
}

/* 垂直排列 */
.flex-column {
  display: flex;
  flex-direction: column;
}
```

### 卡片布局

```scss
.card {
  background-color: #ffffff;
  border-radius: 12px;
  padding: 16px;
  margin: 0 16px 12px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04);
}
```

## 16.7 暗色模式适配思路

当前项目未实现暗色模式，如需适配可通过 CSS 变量实现：

```scss
// 定义亮色主题变量
:root {
  --bg-color: #ffffff;
  --text-color: #333333;
}

// 暗色主题覆盖
@media (prefers-color-scheme: dark) {
  :root {
    --bg-color: #1a1a1a;
    --text-color: #e0e0e0;
  }
}

.card {
  background-color: var(--bg-color);
  color: var(--text-color);
}
```
