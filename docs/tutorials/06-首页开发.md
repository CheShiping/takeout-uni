# 第六章：首页开发

## 6.1 启动页与路由跳转

应用启动后先进入 `pages/index/index.vue` 启动页（Splash），初始化数据后跳转到主页面：

```typescript
// src/pages/index/index.vue
import { useUserStore } from '@/stores/user.store'
import { useShopStore } from '@/stores/shop.store'

const userStore = useUserStore()
const shopStore = useShopStore()

onMounted(() => {
  // 恢复登录态
  if (userStore.isLogin) {
    userStore.fetchUserInfo()
  }
  // 预加载店铺数据
  shopStore.fetchShops()

  // 跳转到主页面
  setTimeout(() => {
    uni.reLaunch({ url: '/pages/main/main' })
  }, 500)
})
```

启动页使用渐变背景 + Logo 居中布局，500ms 后自动跳转。

## 6.2 TabBar 容器页面

`pages/main/main.vue` 是主页面容器，包含 4 个 Tab 和底部导航栏：

```typescript
const tabs = [
  { name: '首页', icon: 'home' },
  { name: '点餐', icon: 'food' },
  { name: '订单', icon: 'order' },
  { name: '我的', icon: 'mine' }
]

const currentTab = ref(0)

function onTabChange(index: number) {
  currentTab.value = index
}
```

### 条件渲染 Tab 内容

使用 `v-if` 而非 `v-show`，切换时销毁/重建组件，节省内存：

```html
<view class="page-content">
  <HomeComponent v-if="currentTab === 0" />
  <FoodCategory v-if="currentTab === 1" :is-food-category-on-show="isFoodCategoryOnShow" />
  <PageOrderList v-if="currentTab === 2" />
  <MineComponent v-if="currentTab === 3" />
</view>
```

### 点餐 Tab 的特殊处理

点餐 Tab（index=1）需要在切换到时触发加载，通过 `isFoodCategoryOnShow` prop 控制：

```typescript
watch(currentTab, (newVal, oldVal) => {
  appStore.setSelectedIndex(newVal)
  if (oldVal !== 1) {
    isFoodCategoryOnShow.value = false
  }
})

function onTabChange(index: number) {
  currentTab.value = index
  if (index === 1) {
    setTimeout(() => {
      isFoodCategoryOnShow.value = true
    }, 50)
  }
}
```

## 6.3 自定义导航栏

main.vue 配置了 `"navigationStyle": "custom"`，隐藏系统导航栏，使用自定义实现。底部 Tab 栏：

```html
<view class="tab-bar">
  <view
    v-for="(tab, index) in tabs"
    :key="index"
    class="tab-item"
    :class="{ active: currentTab === index }"
    @click="onTabChange(index)"
  >
    <image :src="getTabIcon(index)" class="tab-icon-img" />
    <text class="tab-name">{{ tab.name }}</text>
  </view>
</view>
```

激活态样式：文字变为品牌色 `#FF6B35` 并加粗。

## 6.4 首页组件（HomeComponent）

`src/components/home/HomeComponent.vue` 是首页内容，包含三个区域：

### Banner 轮播

```html
<swiper class="banner-swiper" circular autoplay :interval="3000"
  indicator-dots indicator-active-color="#FF6B35">
  <swiper-item v-for="banner in banners" :key="banner.id">
    <view class="banner-item" :style="{ background: getBannerImage(banner.title) }">
      <text class="banner-text">{{ banner.title }}</text>
    </view>
  </swiper-item>
</swiper>
```

使用渐变色作为 Banner 背景，无需图片资源。

### 分类入口

点击入口时根据目标页面选择不同的跳转方式：

```typescript
function onEntryClick(url: string) {
  if (url === '/pages/main/main') {
    uni.reLaunch({ url })       // Tab 页用 reLaunch
  } else {
    uni.navigateTo({ url })     // 普通页用 navigateTo
  }
}
```

### 热销榜单与好评推荐

从后端加载食品数据，按销量/评分排序后取前 6 个：

```typescript
onMounted(async () => {
  const shop = shopStore.shops[0]
  if (shop) {
    const foods = await FoodService.getFoods(shop.id)
    hotRanking.value = foods
      .filter(f => f.monthlySales > 50)
      .sort((a, b) => b.monthlySales - a.monthlySales)
      .slice(0, 6)
    popularRecommend.value = foods
      .sort((a, b) => b.rating - a.rating)
      .slice(0, 6)
  }
})
```

### FoodRecommendCard 组件

每个食品卡片展示图片、名称、价格、月销量，点击跳转到食品详情页。

## 6.5 布局要点

- 使用 `scroll-view scroll-y` 实现整页滚动
- 各区域用白色背景卡片 + 圆角 + 间距分隔
- 食品网格使用 CSS Grid 布局（2 列）
- 底部安全区使用 `env(safe-area-inset-bottom)` 适配

## 小结

本章讲解了首页的开发流程。核心要点是**启动页预加载 + Tab 容器切换 + 组件化拆分**。首页作为门面，需要兼顾加载速度和视觉效果。下一章我们将讲解店铺模块。
